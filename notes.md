# Object Oriented Programming (OOP)

## Characteristics of Objects

- Attributes (Data) and Methods -> dot notation to access methods/ attributes
- Identity/ Memory Address - id()
- Type - type()
- Value -> actual data stored

# Pillars of OOP

- Inheritance - reuse and extend functionality from another class
- Abstraction - hide the implementation details -> show only the necessary functionalities
- Encapsulation - bundling both data (attributes) and behavior (methods) into a single unit.
- Polymorphism - same method -> different behaviours

**object** - an instance of a class
